# MedRemind

**The right pill, the right dose, the right time.**

MedRemind — это умное приложение для контроля приема лекарств, которое помогает не забывать таблетки и следить за здоровьем в реальном времени. Проект ориентирован на пользователей, которым важна дисциплина приема лекарств, а также на их близких или опекунов.

## 🚀 Features

### ⏰ Smart Reminder System
- Напоминание о приеме лекарства в назначенное время.
- После сигнала пользователь подтверждает прием.
- Если подтверждения нет — повторное напоминание через 3 минуты.
- Цикл продолжается до подтверждения.

### 📞 Emergency Contact Alerts
- Если пользователь игнорирует прием, приложение отправляет уведомление (или инициирует звонок) близкому человеку/опекуну.

### 💊 Pill Counter & Database
- Учет оставшихся таблеток по каждому препарату.
- База популярных лекарств.
- Список ближайших аптек с нужными препаратами.
- Быстрый заказ лекарства по назначению врача.

### 📄 OCR Prescription Scanner
- Сканирование рецепта.
- Автоматическое распознавание препаратов и добавление их в план приема/заказ.

### 🎙️ AI Voice Input
- Добавление лекарств и расписания через голосовой ввод (диктовку).

### 👨‍👩‍👧 Caregiver Mode (Parental Control)
- Родственник или опекун может удаленно контролировать прием лекарств и управлять расписанием.

## 📱 Demo build for Expo Go

### Что уже реализовано
- Экран с карточками лекарств и временем приема.
- Подтверждение приема (`Подтвердить`) с автоматическим уменьшением остатка таблеток.
- Кнопка отложенного напоминания на 3 минуты (`+3 мин`).
- Кнопка SOS для эмуляции уведомления опекуну.
- Общая метрика выполнения и счетчик экстренных алертов.
- Блок с демо AI-функциями (OCR / Voice / заказ в аптеке).

### Как запустить в Expo Go
1. Установите Node.js LTS (рекомендуется 20+).
2. Выполните:
   ```bash
   npm install
   npm run start
   ```
3. Установите **Expo Go** на телефон.
4. Отсканируйте QR-код из терминала/браузера Expo.
5. Приложение откроется на устройстве и готово для демонстрации на хакатоне.

## 💡 Idea

Многие люди забывают принимать лекарства вовремя, что может привести к серьезным последствиям. MedRemind решает эту проблему через комбинацию:

- жесткой системы напоминаний;
- контроля со стороны близких;
- автоматизации ввода данных (OCR + AI).
